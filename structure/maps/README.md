# maps

## API
### Map
- Put 设置键值对
- Get 获取对应键的值
- Remove 删除对应key的值
- Keys 获取所有key
- Empty 判断是否为空
- Size 获取map长度
- Clear 清空列表
- Values 获取列表中的所有元素
- String 返回列表的字符串表示
